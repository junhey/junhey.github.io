# 1 清浮动-clear:both

css代码
```css
.layout{}
.left{float:left;}
.right{float:right;}
.clear{clear:both;}
```
html代码
```html
<div class="layout">
    <div class="left">left</div>
    <div class="right">right</div>
    <div class="clear"></div>
</div>
```

# 2 清浮动-overflow:hidden

css代码
```css
.layout{overflow:hidden;zoom:1;}
.left{float:left;}
.right{float:right;}
```
html代码
```html
<div class="layout">
    <div class="left">left</div>
    <div class="right">right</div>
</div>
```

# 3 清浮动-:after

css代码
```css
.layout{}
.clearfix{zoom:1;}
.clearfix:after{
    content:"";
    visibility:hidden;
    display:block;
    clear:both;
    height:0;
    font-size:0;
}
.left{float:left;}
.right{float:right;}
```
html代码
```html
<div class="layout clearfix">
    <div class="left">left</div>
    <div class="right">right</div>
</div>
```
